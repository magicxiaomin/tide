# TideTai

TideTai is a WeChat Mini Program for local sea anglers. It keeps the product simple: show the information clearly, and leave the judgment to the angler.

## Development

Open this repository root in WeChat DevTools. The mini program source lives in `miniapp/`, and CloudBase functions live in `cloud-functions/`.

## Core Rule

Judgment belongs to the user. Information is the product's promise.
