# Launch Checklist

## 代码与配置

- [ ] 微信小程序 AppID 已配置在 `project.config.json` 或微信开发者工具项目设置中。
- [ ] CloudBase 环境 ID 已配置在 `miniapp/config/env.js`。
- [ ] 云函数已部署：`hello`、`setup-db`、`data-today`、`data-forecast`、`spots-nearby`、`catches-create`、`catches-list`。
- [ ] 数据库集合已创建：`users`、`spots`、`user_spots`、`daily_data`、`catch_logs`、`app_config`。
- [ ] 请求域名和云开发权限已在微信公众平台配置。

## 功能验收

- [ ] 首次打开进入 onboarding。
- [ ] 主屏能显示潮汐、风、天气、水温、气压、月相。
- [ ] 七日预报可打开。
- [ ] 钓点列表可选择当前钓点。
- [ ] 渔获记录可保存。
- [ ] 照片可选择、预览、删除，且不上传服务器。
- [ ] 我的渔获可展示历史记录。
- [ ] 渔获详情可打开。
- [ ] 渔获卡可生成并保存到相册。
- [ ] 隐私字段开关会影响渔获卡显示。
- [ ] 主屏完整数据页可打开。

## 弱网与异常

- [ ] 主屏断网时显示缓存数据和明确提示。
- [ ] 云函数失败时展示可读错误，不直接暴露堆栈。
- [ ] 相册权限拒绝时提示用户去设置中开启。
- [ ] 无照片、空军记录、无历史记录均有占位状态。

## 微信审核

- [ ] 小程序简介已填写。
- [ ] 关键词已填写。
- [ ] 隐私政策已配置。
- [ ] 审核截图覆盖主屏、预报、记录、渔获卡、我的渔获。
- [ ] 审核备注说明：不评分、不推荐、不上传照片。

## 上线后

- [ ] 邀请 2-3 位钓友 beta 试用。
- [ ] 至少记录 2 次真实出海流程。
- [ ] 只收敛 Bug，不在 v0.1 审核前追加新功能。
