# Beta Test Plan

目标：在不新增功能的前提下，用真实出海场景验证 MVP 闭环。

## 测试对象

- 自己完成至少 2 次真实出海全流程。
- 邀请 2-3 位钓友做 beta 试用。
- 测试对象优先选择已经有固定钓点、愿意记录渔获、愿意反馈卡点的人。

## 核心路径

必须完整跑通：

打开 -> 看潮汐 -> 出海 -> 回来记录 -> 分享

每次测试记录以下结果：

- 打开主屏是否能在 30 秒内看懂潮汐、风、天气、水温、气压、月相。
- 断网或弱网时是否能看到缓存和清晰错误提示。
- 记录渔获时，鱼种、数量、用饵、备注、照片是否顺手。
- 渔获卡是否能保存到相册，并从相册分享到朋友圈。
- 钓友是否理解「照片只在本机，不上传服务器」。

## Week 9 处理规则

- 只修 Bug，不加新功能。
- 只改影响闭环的问题，不做大重构。
- 任何“推荐、评分、判断今天适不适合钓”的建议都记录为拒绝项，不进入 v0.1。
