# Week 8 分享、隐私与详情收尾

本周完成 MVP 闭环前的产品收尾：

- 渔获卡支持字段隐私开关，可隐藏用饵、水温、气压、月相、风、潮型。
- 渔获卡开启微信分享菜单，并生成不含评分、推荐、判断承诺的自定义分享标题。
- 我的渔获列表可进入渔获详情页，详情页可重新生成该条记录的渔获卡。
- 主屏增加「点击查看完整数据」二级页，集中展示潮汐、风、天气、水温、气压、月相和数据来源。
- 增加 `miniapp/utils/ui.js`，统一错误文案、Toast 与 Loading 包装，减少页面内重复状态处理。

边界确认：

- 分享链路仍然不上传用户照片；朋友圈分享通过保存 Canvas 图片到相册完成。
- 详情页只展示事实数据，不加入评分、推荐或判断。
