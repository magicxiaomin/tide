# Mini Program Submission

## 小程序名称

鲷会 TideTai

## 小程序简介

鲷会是一款给本地海钓玩家使用的出海前数据面板。它集中展示潮汐、风、天气、水温、气压、月相和渔获记录，帮你在出海前快速整理信息，但不替你判断。

## 一句话介绍

出海前 30 秒，看潮汐、风、天气和自己的渔获记录。

## 关键词

海钓、潮汐、天气、风向、水温、气压、月相、渔获记录、渔获卡、钓点

## 类目建议

- 工具
- 生活服务
- 运动户外

最终类目以微信公众平台可选项为准。

## 审核备注

本小程序不提供钓鱼评分、推荐或决策结论，只展示潮汐、天气、风、水温、气压、月相等事实数据，判断由用户自行完成。

用户选择的照片只用于本机 Canvas 合成渔获卡，不上传服务器。服务器只保存结构化渔获记录和天气潮汐快照。

## 截图建议

- 主屏：潮汐波形和数据块。
- 七日预报：每天一行的潮汐与天气概览。
- 渔获记录：鱼种、数量、用饵、照片本地提示。
- 渔获卡：Canvas 合成后的保存相册页面。
- 我的渔获：历史记录与统计。
